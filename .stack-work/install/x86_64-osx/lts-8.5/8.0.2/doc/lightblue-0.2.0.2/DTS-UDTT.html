Syntactic Operations

subst :: Preterm -> Preterm -> Int -> Preterm Source #

Substitution of the variable i in a preterm M with a preterm L "subst M L i" = M[L/i]

shiftIndices :: Preterm -> Int -> Int -> Preterm Source #

shiftIndices m d i add d to all the indices that is greater than or equal to i within m (=d-place shift)

addLambda :: Int -> Preterm -> Preterm Source #

addLambda i preterm: the first subroutine for transvec function, which takes an index and a preterm, transforms the latter in a way that the Var/Appvec with an index j that is equal or greater than i Ex. addLambda 1 (Appvec 0 m) = Appvec 1 (addLambda 1 m) addLambda 0 (Appvec 0 m) = Appvec 0 (App () (Var 1))

deleteLambda :: Int -> Preterm -> Preterm Source #

deleteLambda i preterm: the second subroutine for transvec function, which takes an index i and a preterm p, transforms the latter in a way that the i-th variable vector within p is deleted.

replaceLambda :: Int -> Preterm -> Preterm Source #

replaceLambda i preterm: the third subroutine for transvec function, which takes an index i and a preterm p, transforms the latter in a way that the i-th variable vector within p is replaced by a single variable.

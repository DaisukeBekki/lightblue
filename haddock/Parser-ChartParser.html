data ParseResult Source #

A data type for the parsing result.

Constructors

Full [Node]

when there are at least one node in the topmost box in the chart, returning the nodes.

Partial [Node]

when the parser did not obtain the full result, it collects partially parsed segments from the chart, returning their conjunctions.

Failed

when no box in the chart contains any node.
